# Gerenciador de Tarefas

Projeto desenvolvido para o desafio técnico do Bootcamp Web Front, utilizando Angular no front-end, ASP.NET Core Web API no back-end e SQL Server como banco de dados.

## Tecnologias utilizadas

- Angular
- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- API REST com JSON

## Funcionalidades

- Listar tarefas
- Buscar tarefa por ID
- Criar nova tarefa
- Editar tarefa
- Excluir tarefa
- Persistir dados no SQL Server
- Consumir API no Angular com HttpClient

## Entidade Tarefa

A entidade Tarefa possui os seguintes campos:

- Id
- Título
- Descrição
- Status
- Data de Criação
